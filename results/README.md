# Results Directory

Analysis outputs are written here when the workflow is run locally.

Generated figures, tables, and rendered reports are ignored by Git by default to keep the repository small and easy to upload to GitHub.

Expected subfolders:

- `figures/`: UMAPs, composition plots, signature score plots, and marker feature plots.
- `tables/`: metadata tables, composition summaries, signature scores, pseudo-bulk matrices, and DE results.
- `reports/`: rendered HTML reports.
